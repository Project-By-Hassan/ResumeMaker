Resume Builder Pro — Project Description
Resume Builder Pro is a full-stack AI-powered web application built with Python & Streamlit that allows users to create professional, ATS-optimized resumes in minutes — no design skills needed.

🎯 What It Does
Users fill a step-by-step form (9 sections), choose from 9 professionally designed templates, and instantly download a polished PDF resume. A built-in AI assistant (ResumeGenie) guides them through the process, writes their summary, suggests skills, and reviews their resume in real time.

🛠️ Tech Stack
LayerTechnologyFrontend & BackendPython, StreamlitPDF GenerationFPDF2 (custom templates)AI AssistantGroq API — LLaMA 3.3 70B (free)Data StorageJSON (stats, feedback)DeploymentStreamlit Cloud

✨ Key Features
9 Resume Templates — Modern, Professional, Minimal, Blue Sidebar, Executive, Creative, Tech, Academic, Multi-Column — each with unique header styles and layouts.
Live Preview — Right column updates in real time as user types, showing exactly how their resume will look.
AI Resume Assistant (ResumeGenie) — Powered by LLaMA 3.3 70B via Groq. Writes professional summaries, suggests role-specific skills, improves bullet points, gives ATS tips, and detects the user's name automatically for personalized welcome.
ATS Score — Real-time 0–100 compatibility score that tells users how recruiter-friendly their resume is.
Progress Bar — 9-step visual tracker showing which sections are complete.
Photo Upload — Profile photo embedded directly into PDF.
Admin Panel — Password-protected dashboard showing total resumes generated, today's count, average rating, and recent feedback.
Feedback System — Star rating + comment system with data persistence.
